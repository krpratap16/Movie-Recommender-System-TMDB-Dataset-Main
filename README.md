# 🎬 Movie Recommender System (TMDB Dataset)

This project is a Movie Recommender System built using the TMDB dataset. It suggests movies based on similarity using content-based filtering.

## 🚀 Features
- Recommend movies based on user input
- Uses genres, cast, keywords, and overview
- Cosine similarity for accurate results
- Interactive UI using Streamlit

## 🛠️ Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Streamlit

## ⚙️ How It Works
1. Data preprocessing
2. Feature extraction
3. Vectorization (CountVectorizer)
4. Cosine similarity
5. Recommendation output
